# Inventory-and-Supply-Chain-Management-Analysis
Managing inventory and ensuring an efficient supply chain are critical for business success. In this Power BI project, we dive deep into stock management, demand forecasting, supplier performance, and logistics optimization building a fully interactive dashboard that mirrors real-world operations.

# Project Highlights

🧹 Data Cleaning & Transformation: Used Power Query to clean and model supply chain datasets.
📦 Inventory Metrics: Tracked stock levels, turnover rates, reorder points, and backorders.
🚚 Supply Chain KPIs: Analyzed lead times, delivery performance, and supplier reliability.
🔢 Advanced DAX Calculations: Built actionable insights for procurement and operations.
📊 Professional Dashboard Design: Created an executive-level dashboard with interactive visuals.
🗺️ Real-World Storytelling Techniques: Presented supply chain data effectively for business stakeholders.
